
# Changelog
All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning.

## [0.1.0] - 2025-12-28
### Added
- Antenna-first repo skeleton for 1U CubeSat work.
- Parametric UHF quarter-wave monopole sizing utilities.
- L-network matching synthesis for resistive loads (engineering baseline).
- Frequency sweep CLI for quick sanity checks.
- Documentation pack: requirements, trade study, design, matching/VSWR, deployment, ICD, verification plan.
- Deterministic tests (pytest), linting (ruff), and GitHub Actions CI.
